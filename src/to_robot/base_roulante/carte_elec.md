# Carte électronique pour la base roulante

L'étage éléctrique de votre base roulante doit comporter à minima :
+ Des drivers moteurs 
+ Des encodeurs
+ Une alimentation
+ Des étages de transformation de puissance 
+ Un microcontrolleur
+ Des connectiques (USB,UART)

Exemples :
